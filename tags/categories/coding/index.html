Configure token authentication for git operations through GitHub GitHub has announced token authentication requirements for git operations . They will not support password authentication for git operations and will require using personal access tokens over HTTPS or SSH keys by August 13, 2021. The recommendation is to use personal access tokens and it is explained in this youtube video about personal access tokens .
